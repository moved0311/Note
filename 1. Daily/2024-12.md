Q3. Maximum Subarray Sum With Length Divisible by K
前墜和
```python
class Solution:
    def maxSubarraySum(self, nums: List[int], k: int) -> int:
        acc = list(accumulate(nums, initial=0))
        ans = -inf
        n = len(nums)
        for i in range(k):
            p = inf
            for j in range(i, n + 1, k):
                ans = max(ans, acc[j] - p)
                p = min(p, acc[j])
        return ans
```
