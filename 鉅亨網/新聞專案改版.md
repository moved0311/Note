## Monorepo
```sh
npx create-nx-workspace@latest fe-cnyes
```
## Build
```
npx nx build <project-name>
```
## 新增套件
* 在workspace新增
```
yarn add typescript -DW
```
## 其他
* 打開全部專案關係圖
```
yarn nx graph
```

## 新增專案
* fe-common
```sh
yarn nx g @nx/react:lib fe-common
yarn nx g @nx/react:component section --project=fe-common
```

## 執行專案
```
yarn nx dev <project-name>
```
