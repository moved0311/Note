## Monorepo
```sh
npx create-nx-workspace@latest fe-cnyes
```
## Build
```
npx nx build <project-name>
```
## 新增套件
* 在workspace新增
```
yarn add typescript -DW
```
## 其他
* 打開全部專案關係圖
```
yarn nx graph
```

## 執行專案
```
yarn nx dev <project-name>
```
